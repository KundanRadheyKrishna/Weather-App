## Weather App Description

### Tech Stack Used
- HTML
- CSS
- JavaScript
- Metro Weather API

### What the App Shows (via API)
- Real-time weather of any searched city
- Current weather condition (Sunny, Rainy, Snowy, etc.)

### Why Metro Weather API is Used
The Metro Weather API is used to fetch live and accurate weather data for cities. It allows the app to display real-time weather conditions instantly when a user searches for a city.
